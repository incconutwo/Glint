Glint is a keyboard-first AI assistant for Windows that lives in your system tray. Highlight any text, press a shortcut, and the Ghost HUD appears right where you're typing.

Capabilities

Four modes.
One shortcut away.

⌨️
Ctrl + Alt + G

Grammar Fixer

Instantly correct spelling, grammar, and punctuation while preserving your original tone, formatting, and HTML structure. Works in any app.

teh → the your → you're definately → definitely
💬
Ctrl + Alt + A

Answer Mode

Highlight a question and get a lightning-fast, concise answer in a clean reading window. No tabs, no distractions.

📋
Ctrl + Alt + S

Summarize

Turn long emails, articles, or documents into 3–5 concise bullet points. Read in seconds what took minutes to write.

✨
Ctrl + Alt + P

Custom Prompt

The power tool. Type any instruction: "Make this more professional," "Translate to French," "Rewrite for Twitter." Your rules.

Design

Feels like
Windows built it.

Glint uses the official Windows 11 Mica backdrop, DWM dark mode APIs, and Segoe UI Variable typography. It follows your system theme automatically—light or dark.

  • ✓ Mica frosted-glass transparency
  • ✓ Automatic light/dark mode sync
  • ✓ High-DPI / 4K display support
  • ✓ System tray with synced settings
  • ✓ Instant Mode toggle (skip review)
